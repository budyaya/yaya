# 写作

冰心：“心里有什么，笔下写什么，只听凭此时此刻的思潮自由奔放，从脑中流到指上落到笔尖。微笑也好，深愁也好，洒洒落落，自自然然地画在纸上。”

真诚坦然，不拘形式，不限内容，把心之所想自自然然地画在纸上，会发现作文原本是很惬意的事。

### 【主题】+不仅是……的………，更是……的……

责任不仅是诸葛亮的鞠躬尽瘁死而后已，更是李商隐的春蚕到死丝方尽，是鲁迅的俯首甘为孺子牛。

### 通感

*（视觉）….，仿佛（听觉）
滚滚星河点亮缱绻夜空，仿佛远处传来的缥缈歌谣，令人沉醉入梦。

### 【主题】+多个"一"+对主题的阐述
团结方能共度时艰。"岂曰无衣，与子同袍"。一只口罩，一笔捐款，一句鼓励，一封倡议书，都能解他人的燃眉之急。

在【时间】，谁做什么;在【地点】，谁做什么;在【地点】，谁做什么。+【主题】
在月夜里，苏子泛舟赤壁，几经贬谪，仍凌万顷之茫然;
在草堂中，杜少陵茅屋为秋风所破，却心系天下寒士;
在牢狱中，司马迁椎心泣血，成无韵之离骚。磨难让强者更强，且历久弥坚。

### 【主题】 + 名人名言 + 有人说 + 我认为
成功源于积累。《后汉书》云，“涓流虽寡，浸成江河;烯火虽微，卒能燎原。”但有人说，萤烛之光，难成大器。我却认为，绳锯木断，积水成渊。

### 白描

用动作表情代替形容词，用事实描写事实。

鲁迅：

陈老五送进饭来，一碗菜，一碗蒸鱼；这鱼的眼睛，白而且硬，张着嘴，同那一伙想吃人的人一样。

"那么，我对你说。迅哥儿，你阔了，搬动又笨重，你还要什么这些破烂木器，让我拿去罢。我们小户人家，用得着。"圆规（杨二嫂）一面愤愤的回转身，一面絮絮的说，慢慢向外走，顺便将我母亲的一副手套塞在裤腰里，出去了。

我冒了严寒，回到相隔二千余里，别了二十余年的故乡去。

### 主题造势

不必说 + 也不必说 + 单是

不必说碧绿的菜畦，光滑的石井栏，高大的皂荚树，紫红的桑椹; 也不必说鸣蝉在树叶里长吟，肥胖的黄蜂伏在菜花上，轻捷的叫天子忽然从草间直窜向云霄里去了。单是周围的短短的泥墙根一带， 就有无限趣味。

不用说 + 也不用说 + 单是

不用说落霞与孤鹜齐飞，秋水共长天一色的滕王阁，也不用说春江潮水连海平，海上明月共潮生的春江月夜，单是这一隅的山水草木，已足叫人心旷神怡。
